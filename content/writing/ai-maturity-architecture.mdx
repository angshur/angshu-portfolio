In enterprise settings, the question is shifting from:

> “Which model should we pick?”

to:

> “How does the system work end-to-end?”

### The four architecture questions
1. **Context** — where does truth live?
2. **Tools** — what can the agent reliably do?
3. **Orchestration** — which agent runs when (and why)?
4. **Feedback** — how does the system learn and improve?
